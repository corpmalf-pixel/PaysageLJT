# Structure du projet — PaysageLJT

## Objectif
Avoir un projet propre, clair et simple à maintenir pour ajouter facilement photos et vidéos plus tard.

## Arborescence
- `index.html` → page principale
- `assets/css/styles.css` → styles globaux
- `assets/js/main.js` → scripts
- `assets/media/branding/` → logo, favicon, éléments de marque
- `assets/media/hero/` → photo ou vidéo principale du hero
- `assets/media/services/` → images liées aux services
- `assets/media/realisations/` → photos/vidéos des projets
- `assets/media/about/` → photos de l’entreprise, portraits, camion, etc.
- `content/site-content.md` → infos texte utiles du site
- `README.md` → repères techniques rapides
- `vercel.json` → configuration Vercel

## Règle média
Quand tu ajoutes un fichier, place-le dans le bon sous-dossier.

Exemples :
- `assets/media/hero/hero-jardin-premium-01.jpg`
- `assets/media/realisations/realisation-terrasse-01.jpg`
- `assets/media/realisations/video-chantier-01.mp4`
- `assets/media/about/portrait-dirigeant-01.jpg`

## Avantage
Comme ça :
- toi, tu sais où déposer chaque média
- moi, je peux les retrouver immédiatement
- le projet reste propre, lisible et évolutif
