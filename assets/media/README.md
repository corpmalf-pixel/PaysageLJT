# Organisation des médias

Dépose ici les médias du site pour que l’intégration soit simple et propre.

## Structure
- `branding/` → logo, favicon, éléments de marque
- `hero/` → photo ou vidéo principale de la hero section
- `services/` → images liées aux prestations
- `realisations/` → photos et vidéos des chantiers/projets
- `about/` → photos équipe, portrait, atelier, camion, etc.

## Nommage conseillé
Utilise des noms clairs, sans ambiguïté.

Exemples :
- `hero-jardin-premium-01.jpg`
- `realisation-terrasse-bois-01.jpg`
- `realisation-allee-pavee-02.jpg`
- `video-chantier-01.mp4`
- `logo-ljt-paysage.png`

## Formats conseillés
- Photos : `.jpg`, `.png`, `.webp`
- Vidéos : `.mp4`

## Règle
Quand tu ajoutes un média, garde-le dans le bon sous-dossier.
Comme ça je pourrai le retrouver immédiatement et l’intégrer sans confusion.
